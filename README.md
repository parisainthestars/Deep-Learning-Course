## Covered Topics

- Dimensionality reduction and generative models
- Convolutional neural networks for image classification
- Recurrent models for time series forecasting
- Adversarial robustness and attacks
- Transformers for NLP and computer vision
- Parameter-efficient fine-tuning with LoRA
- Graph neural networks and message passing
- Deep reinforcement learning with DQN and SAC

## Repository Structure

Each homework folder contains its own notebook/code, figures when available, and a dedicated `README.md` explaining the implementation, results, and main takeaways.
